Grammatica

Grammatica is a C# and Java parser generator (compiler compiler)

View on GitHub Download .zip Download .tar.gz

Grammatica improves upon simlar tools (like yacc and ANTLR)...

  • by creating well-commented and readable source code,
  • by having automatic error recovery and detailed error messages,
  • and by support for testing and debugging grammars without generating source code.

See the wiki page Why use Grammatica? for reasons to use or avoid Grammatica.
